## Jhipster core knowledge

- **CLI Help**
  - jhipster --help
    - it gives a list of subcommands and their purposes
    - e.g. jhipster aws deploys the current app to aws
  - **CLI subcommand help**
    - jhipster subcommand --help
    - e.g. jhipster aws --help
  - **Key flags**
    - -d debug
    - -V version
